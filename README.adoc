= README.adoc

Install MCP on Codex:

[,bash]
----
./data-translator-mcp/install-on-codex.sh
----

Check the MCP installation:

[,bash]
----
codex mcp list
grep -A 3 ~/.codex/config.toml
----

Install the required dependencies in the projects:

[,bash]
----
(cd data; npm i)
(cd data-translator-mcp; npm i)
----

To rebuild all generated data, run:

[,bash]
----
rm -rf data/events/generated
----

Run Codex:

[,bash]
----
codex
----

Send the following prompts:

* Prepare translations
** After this prompt, you can check (with `git status`) if some files in data/generated were changed.
** In this phase, the /tmp/payload_input.json will be generated by the tool `prepare_translations`.
** Check the translations manually and then...

* Apply translations
** This will call the tool `apply_translations`
** In this phase, the files data/events/*.ts will be generated.

* Build dataset
** This will call the tool `build_dataset`
** In this phase, the file data/events/events.json will be generated.
